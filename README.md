# conexao-verde
# 🌿 Conexão Verde

Conexão Verde é um aplicativo voltado à sustentabilidade que conecta pessoas interessadas em causas ambientais com projetos locais, como plantio de árvores, hortas urbanas e ações de reflorestamento.
